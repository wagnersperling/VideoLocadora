1 - Ferramentas utilizadas

- Visual Studio Community 2017
- SQL Server 2017
- SQL Server Management Studio v17.7

2 - Instruções

- Executar os comandos do arquivo "banco de dados.txt"  no  SQL Server Management Studio para gerar o banco de dados.
- Na classe QueryDapper do projeto RepositoryDapper, no App.Config do projeto RepositoryEntity e no Web.config do projeto VideoLocadora, altere "source=WAGNER\SQLEXPRESS" na string de conexão pelo valor no qual corresponde o nome do servidor de vocês.
- Execute o projeto no Visual Studio e faça uso da aplicação.
